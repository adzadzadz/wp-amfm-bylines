# AMFM Bylines Plugin

## Description
The AMFM Bylines Plugin is a custom WordPress plugin designed to enhance the functionality of your website. It provides additional features and customization options to improve user experience and streamline content management.

## Features
- **Rankmath Extended Features for Bylines**: Enhance your SEO capabilities with extended support for bylines.
- **Custom Profile Schema**: Add structured data for profiles to improve search engine visibility.
- **Elementor Integration**: Seamlessly integrate with Elementor for advanced design capabilities.
- **Shortcode Helpers for ACF**: Simplify the use of Advanced Custom Fields with shortcode helpers.

## Installation
1. Download the plugin files.
2. Upload the plugin folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
